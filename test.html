Quick Start Guide

  1. Start Local Server: Open terminal in this folder and run: python -m http.server 8000
  2. Open App: Go to http://localhost:8000
  3. Test on Phone: Use your phone's browser or scan QR code from your computer
  4. Install PWA: Add to home screen for app-like experience

Mobile Testing Tips

  • Use Chrome DevTools Device Mode to simulate mobile screens
  • Test touch interactions and responsive design
  • Verify PWA installation on actual mobile devices
  • Check offline functionality by disabling network
